HTML & CSS Basics Summary

HTML Summary

HTML(Hyper Text Markup Language) is used to define our page's content, structure and meaning.
You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses 'elements' to describe(anotate) content
  • HTML elements typically have an opening tag, content followed by the closing tag
  • You can also have the void(empty) elements like images
  • You can also configure elements with attributes
  • There's a long list of HTML elements, but you'll gain experience overtime so no worries;)

Learn more about the HTML elements on the MDN HTML element reference

CSS Summary

CSS(Cascading Style Sheets) is used for styling your page's content.

  • Styles are assigned via property-value pairs
  • You can assign value via the 'style' attribute
  • To avoid code duplication, you typically use global styles(e.g. via the style element instead)
  • Alternatively, you can work with the external stylesheet files which you 'link' to
  • When working with CSS, concepts like 'inheritance', 'specificity' and the 'box-model' should be understood

Learn more about the CSS elements on the MDN CSS element reference
